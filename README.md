# Special Partners 회사 소개 페이지 목업

**의료기기 인허가 맞춤형 컨설팅 전문 파트너**

Special Partners의 회사 소개 페이지를 위한 12가지 다른 스타일의 목업 디자인입니다.

## 🎨 목업 스타일

### 목업 #1 - 모던 미니멀 디자인
- **특징**: 깔끔하고 심플한 레이아웃
- **색상**: 흑백 기반의 미니멀한 디자인
- **구성**:
  - 히어로 섹션 (의료기기 인허가 전문 파트너)
  - 통계 정보 (100+ 인허가 승인, 3개국 대응, 100% 심사 통과율)
  - 회사 소개
  - 주요 서비스 (사이버보안 시스템 구축, 글로벌 인허가 컨설팅, 규제 준수 교육)
  - 핵심 가치 (실질적 구축, 최신 트렌드, 맞춤형 솔루션)
  - 연락처 섹션

### 목업 #2 - 다이나믹 그라데이션 디자인
- **특징**: 생동감 있는 색상과 대담한 타이포그래피
- **색상**: 블루, 시안, 퍼플 그라데이션
- **구성**:
  - 동적 히어로 섹션 (의료기기 인허가 전문 메시지)
  - 특징 카드 (실질적 구축, 글로벌 전문성, 100% 승인율)
  - 성과 통계 (100+ 인허가 승인, 3개국 대응)
  - 미션 및 역량 시각화 (실질적 시스템 구축 강조)
  - CTA(Call to Action) 섹션

### 목업 #3 - 프로페셔널 기업 디자인
- **특징**: 신뢰감 있고 전문적인 기업 이미지
- **색상**: 블루 기반의 비즈니스 컬러
- **구성**:
  - 전문적인 히어로 섹션 (의료기기 인허가 전문)
  - 회사 소개 (글로벌 규제 전문성)
  - 사업 분야 (사이버보안 시스템 구축, 한국/미국/유럽 인허가)
  - 경영진 소개
  - 문의 폼이 포함된 연락처 섹션

### 목업 #4 - 테크 전문가 디자인
- **특징**: 기술 중심의 사이버보안 전문 이미지
- **색상**: 다크 테마 + 사이버 블루/시안 컬러
- **참고**: 기술 스타트업, 사이버보안 전문 기업
- **구성**:
  - 기술적 히어로 섹션 (실시간 규제 준수 현황 대시보드)
  - 그리드 배경 효과로 기술적 전문성 강조
  - 핵심 서비스 (시스템 구축, 글로벌 인허가, 지속적 준수)
  - 체계적인 컨설팅 프로세스 (4단계)
  - 국가별 규제 준수 상세 정보 (한국/미국/유럽)
  - 데이터 시각화 및 기술 스펙 강조

### 목업 #5 - 글래스모피즘 3D 디자인 (NEW!)
- **특징**: 반투명 유리 효과의 프리미엄 디자인
- **색상**: 라이트 테마 + 그라데이션 (보라-블루)
- **참고**: Apple, Microsoft 등 글로벌 테크 기업
- **구성**:
  - 글래스모피즘 효과 (backdrop-blur, 반투명 카드)
  - 3D 플로팅 카드 애니메이션
  - 실시간 규제 준수 현황 (진행률 바)
  - 프리미엄 서비스 섹션
  - 4단계 프로세스 시각화
  - 모던하고 세련된 UI/UX

### 목업 #6 - 데이터 대시보드 디자인 (NEW!)
- **특징**: 실시간 데이터 분석 및 시각화 중심
- **색상**: 화이트 베이스 + 컬러풀 차트
- **참고**: Tableau, Google Analytics, 데이터 시각화 플랫폼
- **구성**:
  - KPI 대시보드 (100+ 승인, 3개국, 50+ 제조사, 100% 통과율)
  - 실시간 활동 타임라인
  - 국가별 규제 준수 현황 (진행률 바 차트)
  - 월별 트렌드 차트 (인터랙티브 바 차트)
  - 서비스 분야별 성과 (3개 카테고리)
  - 데이터 중심의 투명한 정보 제공

## 🚀 시작하기

### 필수 요구사항
- Node.js 20.0.0 이상
- npm 또는 yarn, pnpm

### 설치

```bash
# 의존성 설치
npm install
# 또는
yarn install
# 또는
pnpm install
```

### 개발 서버 실행

```bash
npm run dev
# 또는
yarn dev
# 또는
pnpm dev
```
